What we do

Python

Python is a relatively new language, but it's become super popular with everything from building applications to machine learning! In this course we take a look at:

  • Variables
  • Functions
  • Object oriented programming

NodeJS

NodeJS is a variation of JavaScript that's used for programming servers, which allows for more security and data sharing between users! In the events, we created a bakery application that can:

  • Get all the bakery items (different kinds of cakes, pies, etc.) in the database
  • Filter the list of items by item category and name
  • Add custom items to the database
  • Delete items from the database

Web design

Our latest and greatest events! We learn the basics of HTML, CSS, and good web design techniques through example projects.
